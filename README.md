## Tech Stack

**Client:** NextJS, TailwindCSS

**Server:** NodeJS, ExpressJS, MySQL, Prisma


## Installation

Activate the Server with development mode using nodemon

```bash
  cd server
  npm Install
  npx nodemon
```
